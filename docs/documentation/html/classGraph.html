Detailed Description

template<class T>
class Graph< T >

Class representing a directed graph.

The Graph class defines a directed graph structure using vertices and edges. It supports various graph operations like adding/removing vertices, edges, performing depth-first search (DFS), breadth-first search (BFS), topological sorting, etc.

◆ setupInDegreeAndOutDegree()

template<class T >
void Graph< T >::setupInDegreeAndOutDegree

Sets up the inDegree and outDegree of vertices in the graph.

Calculates and assigns the inDegree and outDegree for each vertex in the graph. The inDegree of a vertex is the count of incoming edges, while the outDegree is the count of outgoing edges. This function iterates through all edges in the graph to update the inDegree and outDegree of each vertex. Note: This function assumes that the graph has been appropriately constructed with vertices and edges.
